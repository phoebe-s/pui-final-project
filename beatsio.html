Beats.io

Single-player computer game utilizing AI

Spring 2018, Fundamentals of Programming

Tools: Python, Pygame

Beats.io is an Agar.io-style single-player game that focuses on the user's mouse movements to navigate the screen. The goal of the user is to eat food dots to grow big enough to eat an artificially intelligent boss dot before it does the vice versa. In addition, food dots blink to the beats of selected music for an enhanced user experience.

The Agar.io variation comes with an AI boss and camera focus that centers a game object relative to the user's mouse and utilizes audio beat detection using a Pygame module to spawn game objects. I implemented object-oriented programming to achieve clean, modular code.
